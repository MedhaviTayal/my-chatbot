# My Python Chatbot

A simple terminal-based AI chatbot built with Python and the OpenAI API.

## Features
- Conversational memory (remembers context within a session)
- Clean terminal interface
- Easy to customize the system prompt

## Setup

1. Clone the repo:
```bash
   git clone https://github.com/YOUR_USERNAME/my-chatbot.git
   cd my-chatbot
```

2. Create a virtual environment and install dependencies:
```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install openai python-dotenv
```
